# Game Controls

## 

##### Updated October 20, 2025 (Version 13.350)

## Overview

Foundry Virtual Tabletop provides a variety of keyboard shortcuts for interacting with different pieces of the software in different contexts. This article provides a detailed list of all detail the keyboard shortcuts and their related functions.

You can configure the keybindings used for these controls by clicking the `Configure Controls` button in the Game Settings () sidebar tab. For more information, see [Keybinds](/article/keybinds/ "Keybinds").

Due to technical limitations, mouse interactions (like right-clicking) cannot be customized.

The Configure Controls Application.

## Default Controls

This form describes default keyboard and mouse commands used to control various aspects of the Foundry Virtual Tabletop environment. Some controls are only available to a GameMaster user, those are highlighted in light purple.

If you are using macOS, any reference to Ctrl in this document should be assumed to mean Command.

## Specialized Controls

These controls apply only during specific interactions.

### Token Controls

1. #### Open Character Sheet

   C
2. #### Set (non-Owned) Token as Target

   T, or Double Right-Click
3. #### Place Hidden Token

   Alt + Drop
   when creating a Token.

### Token Drag Movement Controls

1. #### Toggle Distance Ruler

   R
2. #### Place Ruler Waypoint

   R
3. #### Ascend Token Vertically

   Numpad + while dragging a token  
   Add SHIFT to ascend a full grid unit
4. #### Descend Token Vertically

   Numpad - while dragging a token  
   Add SHIFT to descend a full grid unit
5. #### Toggle Unconstrained Movement

   U
6. #### Cycle Movement Method

   Tab while dragging a token

### Walls Layer Controls

1. #### Chain Wall Creation

   Hold Ctrl
2. #### Select Chained Segments

   Alt + Left Click
3. #### Open and Lock Doors Silently

   Alt + Left Click

### Tile Controls

   Alt + Drop
   when placing a Tile.

### Template Controls

   Alt + Drop
   when placing a Template.
