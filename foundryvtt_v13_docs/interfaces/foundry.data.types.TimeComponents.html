Interface TimeComponents

A decomposition of the integer world time in seconds into component parts. Each component expresses the number of that temporal unit since the time=0 epoch.

interface TimeComponents {
    day: number;
    dayOfMonth: number;
    dayOfWeek: number;
    hour: number;
    leapYear: boolean;
    minute: number;
    month: number;
    season: number;
    second: number;
    year: number;
}
