Class Playlists

The singleton collection of Playlist documents which exist within the active World. This Collection is accessible within the Game object as game.playlists.

See

  • foundry.documents.Playlist: The Playlist document
  • foundry.applications.sidebar.tabs.PlaylistDirectory: The PlaylistDirectory sidebar directory
